# Contributing to this project

We love pull requests!

## Contributor License Agreement

Contributions to this project must be accompanied by a Contributor License Agreement (CLA). You (or your employer) must keep the agreement on file with us, which you can sign at <https://cla.developers.google.com/>.

When you submit a Pull Request, a CLA bot will automatically check if you have signed the CLA and flag the PR appropriately (e.g., build success, missing signature). Simply follow the instructions provided by the bot. You only need to do this once across all repositories using our CLA.

## Code Reviews

All submissions, including submissions by project members, require review. We use GitHub pull requests for this purpose. Consult [GitHub Help](https://help.github.com/articles/about-pull-requests/) for more information on using pull requests.

## Community Guidelines

This project follows the [Google Open Source Community Guidelines](https://opensource.google/conduct/).
